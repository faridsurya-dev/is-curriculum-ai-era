# Welcome

This is your new notes site. It already works — now make it yours.

## How it's organized

- Notes live in the `vault/` folder of this repository.
- Every `.md` file becomes a page. This note is the home page.
- Link notes together: [[First Note]] — and the connection shows up in the graph.

## What's next

1. Open [[How to Use]] — the essentials are there.
2. Rename the site in `markdown-publish.config.json` (the `siteName` key).
3. Write notes — the site rebuilds itself on every change.
